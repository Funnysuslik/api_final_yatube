## Setup
To run this project, install it locally using npm:

```
$ cd ../api_final_yatube
$ py venv vnev
$ . venv/Scrpits/Activate (. bin/Scripts/Activate)
$ pip install -r 'requarements.txt'
$ cd yatube_api
$ py manage.py runserver
```
